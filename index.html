¿Como se calculo?

  • Con el método Holliday-Segar:
  • De 0kg a 10kg, se calcula 100cc por cada kilo.
  • Se suman 50cc por cada kilo de peso por arriba de 10kg, hasta 20kg
  • De 20kg para arriba, se suman 20cc por cada kilo adicional
  • Con el método de Superficie Corporal :
  • Cuando el niño pesa mas de 30kg
  • Se multiplica SC (Superficie Corporal) por 2000 o 1500 para hallar el volumen diario
  • Superficie Corporal es igual a ( (peso * 4) + 7) / (peso + 90)
